Yes. Run it once per landing page — there's no limit on how many pages you build.

No. The Shopify files are ready to copy-paste. The installation guide walks you through exactly where each file goes in the theme code editor — it takes about 10 minutes.

Will the page actually look like my brand?

Yes — this is what makes it different from other landing page tools. The system fetches your live website and extracts your brand's visual identity (colours, fonts, spacing, button style) before building anything. The preview will look like a page that already exists on your site.
